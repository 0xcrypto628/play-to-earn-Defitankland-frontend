## Run Locally

Clone the project

```bash
  git clone https://github.com/yellowflash95628/play-to-earn-Defitankland-frontend.git
```

Go to the project directory

```bash
  cd my-project
```

Install dependencies

```bash
  yarn install
```

Start the server

```bash
  yarn start
```
